## 实验记录

- 杨氏模量：流程较基础。
- 声速的测量：步骤较多，记录过程要完整。
- 表面张力：测量量较集中。
- 迈克尔孙：调节环节较多，提前熟悉流程更方便。
- 分光计：实验过程依赖仪器调节。
- 模拟法测电场：现场绘图和数据整理工作量较大。
- 密立根：观察与筛选油滴过程较长。
- 理想气体：手动操作较多。
- 光电效应：自动化程度较高。
- 太阳能电池：以数据记录与整理为主。
- 自组望远镜与显微镜：仪器调节需要时间。
- 磁场的测量：以测量和记录为主。
