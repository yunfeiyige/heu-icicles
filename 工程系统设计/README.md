# 工程系统设计

## 目录说明
- `Arduino`：Arduino 相关代码与库文件。
- `PPT`：课件与讲义。
- `TRIZQFDAHP`：课程报告、分析表格与相关资料。

## 课程内容

本课程资料主要覆盖以下部分：

- 设计与分析：市场调研、QFD、AHP、TRIZ 等报告材料。
- 机械基础：课程配套理论内容。
- 软件建模：SolidWorks 相关材料。
- Arduino 编程：小车控制相关代码与开发环境资料。

## Arduino 说明

- 目录中的部分示例代码依赖 `IRremote 2.6.0` 及以下版本。
- 同一 Arduino 工程目录中只保留一组 `setup()` 与 `loop()`。
- 安装库路径：`Sketch -> Include Library -> Manage Libraries`。
