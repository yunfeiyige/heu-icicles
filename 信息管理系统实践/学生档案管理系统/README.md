# KEST - 一个学生档案管理系统

## 现有文件

- `data_manage.zip` 与 `data_manage.z01` 至 `data_manage.z06`：分卷压缩包。
- `学生档案管理系统.docx`：项目说明文档。

## 功能说明

- 用户管理
- 学生信息的增删查改
- 登录页背景与界面效果配置

## 使用方式

1. 将全部分卷压缩包放在同一目录下并解压 `data_manage.zip`。
2. 启动后端：在解压后的目录中执行 `npm start`。
3. 启动前端：在解压后的 `fronted` 目录中执行 `npm run dev`。
4. 在浏览器中访问系统页面。
